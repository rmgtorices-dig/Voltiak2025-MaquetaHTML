# Propuesta HTML/CSS - Voltiak 2025

Esta es una propuesta de maqueta HTML/CSS puro para el proyecto Voltiak 2025.

## Estructura del proyecto

### Carpetas de páginas y componentes

Cada carpeta tiene el nombre de la página o componente a usar. En cada una se encuentra un archivo HTML y un CSS correspondiente:

- **about/** - Página de información sobre la empresa
- **cart/** - Página del carrito de compras
- **contact/** - Página de contacto
- **footer/** - Componente del pie de página
- **header/** - Componente del encabezado
- **home/** - Página de inicio
- **store/** - Página de la tienda
- **store-slug/** - Página de detalle de producto

### Recursos

- **imgs/** - Carpeta que contiene las imágenes usadas como propuesta

### Estilos globales

- **global.css** - Archivo CSS que contiene estilos para quitar y limpiar los estilos predefinidos de los navegadores
